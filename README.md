# Plataforma de Experiencias Turísticas Sostenibles de Aysén

Esta carpeta contiene una propuesta de plataforma web para difundir y reservar experiencias turísticas sostenibles en la Región de Aysén.

## Qué incluye
- Diseño semántico con header, nav, main, section y footer.
- Página responsive con CSS externo.
- Tarjetas de experiencias generadas dinámicamente desde un arreglo en JavaScript.
- Filtros por categoría con interacción mediante `classList`.
- Botones para mostrar u ocultar la descripción de cada experiencia.
- Formulario de reserva con validación HTML y JavaScript.
- Descuento automático de cupos tras una reserva válida.

## Archivos
- `index.html`: estructura principal de la página.
- `styles.css`: estilos visuales y responsive.
- `script.js`: lógica de datos, filtros, tarjetas y formulario.

## Cómo abrirla
Abre el archivo `index.html` en tu navegador o sirve la carpeta con un servidor simple como:

```bash
python3 -m http.server 8000
```

Luego visita `http://localhost:8000`.
